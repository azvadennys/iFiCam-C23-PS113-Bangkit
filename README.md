## Tech stack & Open-source libraries
- Minimum SDK level 21
- 100% Kotlin
- MVVM Architecture
- [Firebase](https://firebase.google.com/docs/database?hl=id) - Firebase Authentication
- [Google Map SDK](https://developers.google.com/maps/documentation/android-sdk/overview?hl=id) - Provide map and location service for Android App.
- [Retrofit2 and Okhttp](https://github.com/square/retrofit) - Construct the REST APIs.
- [uCrop](https://github.com/Yalantis/uCrop) from Yalantis - to Crop Image
- [Lottie](https://lottiefiles.com/97952-loading-animation-blue) - Loading Animation

  - Uploading stories
  - Open the map to find out the location of each uploader
  - Detecting fish
